How does git help you keep track of changes?

Git is a program that allows users to work on multiple versions of the same file. Those versions can be submitted to peers for review, then merged into the master copy of the project. People can work on different portions of a file at the same time, then combine them. This saves a lot of time. Also, you can see what changes were made and when they were made, which aids tremendously in troubleshooting.

Why use GitHub to store your code?

GitHub allows thousands of programmers to view/edit the source code for thousands of programs, enabling immense amounts of collaboration and increased productivity for free. Other version control systems are not as popular and/or cost money. If the benefits are not immediately clear, they will be when robots are doing all our chores.
